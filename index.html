Projeto Construtora Fictícia - Grupo NEWBIE

apresentação acadêmica de desenvolvimento web e design visual. Um estudo de caso focado em arquitetura moderna e experiência digital para o setor de construção civil

Sobre o Projeto

Este projeto será desenvolvido como parte integrante da grade curricular de Desenvolvimento Web. O objetivo principal é criar uma plataforama robusta para uma empresa fictícia do ramo de construção, aplicando conceitos de design responsivo, acessibilidade e SEO.

A proposta visual busca transmitir solidez e transparência com o cliente e características essenciais para o mercado imobiliário.

Tecnologias & Contexto

Front-end: HTML, CSS, JavaScript

Metodologia: Desenvolvimento baseado em boas práticas de programação, organização do código e planejamento visual da interface.

Acadêmico: Projeto do primeiro semestre de 2026 - Desenvolvimento Web
