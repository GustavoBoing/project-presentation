##Aprsentação de Projeto Programação Web

Este projeto foi desenvolvido com o objetivo de apresentar uma proposta visual e interativa para uma empresa do setor de construção civil, utilizando tecnologias fundamentais do desenvolvimento web front-end.

#📌 Tecnologias utilizadas
HTML5
CSS3
JavaScript

##🎯 Objetivo do projeto

Criar uma interface moderna, responsiva e intuitiva para apresentação institucional de uma construtora fictícia, destacando informações sobre serviços, valores da empresa e interação com o usuário.

##⚙️ Funcionalidades implementadas
Estruturação semântica com HTML
Estilização responsiva com CSS
Interações dinâmicas com JavaScript
Navegação visual entre seções
Componentes informativos organizados em cards

##💡 Recursos adicionados com JavaScript
Interatividade nos elementos da página
Comportamentos dinâmicos de navegação
Manipulação de elementos do DOM
Melhor experiência visual para o usuário
